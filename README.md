# Data Science Jobs & Salary Analysis

## Overview
Analysis of 9,355 data science job listings worldwide.

## Key Findings
1. Analytics Engineering Manager is highest paid ($399K avg)
2. Executive level earns 2x more than Entry level ($189K vs $88K)
3. In-person roles pay more than remote ($155K vs $144K)
4. Data Science and Research is the most common category
5. Machine Learning and AI is the fastest growing category

## Tools Used
Python, pandas, matplotlib, seaborn, Jupyter

## Dataset
Source: Kaggle - Jobs and Salaries in Data Science
Rows: 9,355 | Columns: 12
